| **Section** | **Description** |

| **Section** | **Description** |
|--------------|-----------------|

| **Section** | **Description** |
|--------------|-----------------|

## Day 04 – Exploratory Data Analysis (EDA)

| Section | Description |
|--------------|-----------------|

| **Topics Covered** | Dataset Exploration, Univariate Analysis, Bivariate Analysis, Multivariate Analysis, GroupBy, CrossTab, Summary Statistics, Correlation Analysis. |
